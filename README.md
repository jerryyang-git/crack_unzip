# crack_unzip
使用python编写基于7z Cmd命令的压缩包密码工具

运行后直接托入文件回车  
程序运行根据当前PC核心多线程测试密码  
测试完成即结束，若要中途终止，直接关闭终端窗口即可

内置拉取[galgame-Password.txt](https://github.com/jerryyang-git/compressed_password_book/blob/main/galgame-Password.txt) 密码文件 **选项** 或者自定义txt文件
